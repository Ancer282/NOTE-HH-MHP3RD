#	HH NOTES ON SCREEN

Mods to display hh notes on screen

## REQUISITOS

[ARMIPS.EXE BY Kingcom](https://www.romhacking.net/utilities/635/)

[MODMANAGER BY KUROGAMI](https://gamebanana.com/tools/19380)

## HOW TO COMPILE

Place armips.exe in the folder and run

 [hd]
 armips.exe hd.asm

 [nohd]
 armips.exe nohd.asm
